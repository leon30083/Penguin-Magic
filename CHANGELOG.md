# 更新日志 Changelog

本文件记录企鹅工坊的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 完善开源项目规范管理
  - 优化 .gitignore 规则，添加可执行文件、临时文件、备份文件等忽略规则
  - 为空目录添加 .gitkeep 文件，确保目录结构完整
  - 统一前后端版本号为 0.2.4
  - 创建 CHANGELOG.md 和 CONTRIBUTING.md 文档

## [0.2.4] - 2025-01-04

### 新增
- 导入创意库功能：支持从 https://opennana.com/awesome-prompt-gallery/ 导入创意
- 后端服务状态指示器：红房子/黄房子图标显示后端服务运行状态

### 改进
- 支持多并发生成图片

### 问题修复
- 修复部分场景下后端服务意外中断的问题（持续排查中）

## [0.2.3] - 2025-01-01

### 新增
- 最新分支支持多并发图片生成

## [0.2.2] - 2024-12-XX

### 改进
- 性能优化和稳定性提升

## [0.2.1] - 2024-12-XX

### 改进
- 界面优化
- Bug 修复

## [0.2.0] - 2024-12-XX

### 新增
- 桌面级管理体验
- 智能文件夹和叠放功能
- 创意库系统
- 完整的创作闭环

### 改进
- 重构桌面交互逻辑
- 优化拖拽体验

## [0.1.0] - 2024-11-XX

### 新增
- 初始版本发布
- 基础的 AI 图像生成功能
- 简单的图片管理功能

---

## 版本号说明

- **主版本号 (MAJOR)**：不兼容的 API 修改或重大架构变更
- **次版本号 (MINOR)**：向下兼容的功能性新增
- **修订号 (PATCH)**：向下兼容的问题修正

## 变更类型

- **新增**：新功能
- **改进**：对现有功能的改进
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **问题修复**：Bug 修复
- **安全**：安全相关的修复
