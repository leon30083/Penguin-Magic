## Document Summaries

### mcp-servers.md

Complete guide for configuring 10 MCP servers:
- **Memory**: Knowledge graph for project architecture
- **GitHub**: Repository, PR, Issue management
- **Context7**: Latest tech documentation queries
- **Chrome DevTools**: UI automation testing
- **Sequential Thinking**: Complex problem analysis
- **Web Search Prime**: Web search with Chinese optimization
- **Web Reader**: Web page content extraction
- **ZRead**: Open source repository reading
- **4.5v Image Analysis**: Advanced image understanding
- **ZAI MCP**: UI-to-code, error diagnosis, diagram understanding

Includes:
- Server installation commands
